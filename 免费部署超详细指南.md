## 🚀 第一步：注册 GitHub 并上传代码

### 1.1 注册 GitHub 账号
1. 打开 https://github.com
2. 点击右上角 "Sign up"
3. 按提示输入邮箱、密码、用户名
4. 完成注册！

### 1.2 创建一个新仓库
1. 登录 GitHub 后，点击右上角 "+" 号
2. 选择 "New repository"
3. Repository name（仓库名）填：`zhisheng-platform`
4. 选择 Public（公开）
5. 点击 "Create repository"

### 1.3 上传代码

别急！现在你需要先拿到我们整理好的部署文件。
我会把两个文件夹（`vercel-deploy` 和 `render-deploy`）打包给你，
你解压缩后，会有两个文件夹。

---

## 🚀 第二步：部署后端（Render）

### 2.1 注册 Render 账号
1. 打开 https://render.com
2. 点击 "Sign Up"
3. 选择用 GitHub 账号登录（最方便）

### 2.2 创建后端服务
1. 登录后，点击右上角 "+ New" → "Web Service"
2. 选择 "Build and deploy from a Git repository"
3. 点击 "Connect account" 连接你的 GitHub
4. 选择你刚才创建的 `zhisheng-platform` 仓库
5. 按下面配置：
   - Name: `zhisheng-backend`（名字自己喜欢就行）
   - Region: 选离你最近的，比如 `Singapore`
   - Branch: `main`
   - Root Directory: `render-deploy`（重点！）
   - Runtime: `Node`
   - Build Command: `npm install`
   - Start Command: `node index.js`
6. 点击 "Create Web Service"
7. 等待几分钟，部署成功后会有一个网址，类似：
   `https://zhisheng-backend.onrender.com`
   
**✅ 记下来这个网址！** 后面要用！

---

## 🚀 第三步：部署前端（Vercel）

### 3.1 注册 Vercel 账号
1. 打开 https://vercel.com
2. 点击 "Sign Up"
3. 选择用 GitHub 账号登录

### 3.2 修改前端配置
在你下载的 `vercel-deploy/src/services/api.ts` 文件里，
把第 8 行的地址改成你刚才 Render 的地址：
```javascript
return import.meta.env.VITE_API_URL || 'https://你的render地址.onrender.com';
```

### 3.3 部署前端
1. 登录 Vercel 后，点击 "Add New..." → "Project"
2. 选择你的 GitHub 仓库 `zhisheng-platform`
3. 按下面配置：
   - Framework Preset: 选 `Vite`
   - Root Directory: `vercel-deploy`（重点！）
4. 点击 "Environment Variables"（环境变量），添加：
   - Name: `VITE_API_URL`
   - Value: 你的 Render 后端地址（比如 `https://zhisheng-backend.onrender.com`）
5. 点击 "Deploy"
6. 等待几分钟，部署成功后会给你一个网址！
